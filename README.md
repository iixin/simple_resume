# 简约简历模板
所见即所得的简历编辑工具

浏览器所见与打印PDF后效果一致

## 快速开始

1. 根据example.json, 新建并编辑 resume.json 文件

2. 在 VS Code 打开本文件夹, 并点击 "Go Live" 运行 "Live Server" 扩展（需要先安装该扩展）, 会自动在浏览器打开
   
3. 复制地址栏链接并使用 Microsoft Edge 打开（可以使导出PDF中的链接可点击）, 打印,  打印选项如下:
   - 打印机: 另存为PDF
   - 布局: 纵向
   - 页面: 全部
   - 纸张大小: A4
   - 缩放: 100
   - 每张纸打印的页数: 1
   - 边距: 无
   - 选项: 背景图形
